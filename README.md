# Smart Leads Dashboard

Lead management dashboard built with React + TypeScript on the frontend and Express + TypeScript on the backend. Includes JWT auth, role-based access, filters, pagination, and CSV export.

## Quick links

- Setup: docs/setup.txt
- API: docs/api.txt

## Notes

- Roles: admin (full access + export), sales (create/update/view).
- Pagination limit: 10 per page.
